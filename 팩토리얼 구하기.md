# 팩토리얼 구하기
> 1부터 n까지의 곱


```
1! = 1
2! = 2 * 1
3! = 3 * 2 * 1

0! == 1
```

* For문을 이용한 팩토리얼
 
```
# 연속한 숫자의 곱을 구하는 알고리즘
# 입력: n
# 출력: 1부터 n까지의 연속한 숫자를 곱한 값

def fact(n):
    f = 1                           # 곱을 계산할 변수, 초기값은 1
    for i in range(1, n + 1):       # 1부터 n까지 반복
        f = f * i                   # 곱셉 연산으로 수정
    return f
    
print(fact(1))
print(fact(5))
print(fact(10))

===실행 결과===
1
120
3628800    
```
