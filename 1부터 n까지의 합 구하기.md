# 1부터 n까지의 합 구하기

* 문제
	* 1부터 n까지의 연속한 숫자의 합 구하기

* 입력
	* n까지에 해당하는 n

* 출력
	* 1부터 n까지의 합에 해당하는 숫자

```
import math # 수학 모듈 사용

#절댓값 알고리즘 1(부호판단)

# 입력:실수 a
# 출력:a의 절댓값

def abs_sign(a):
    if a >= 0:
        return a
    else:
        return -a
    
# 절댓값 알고리즘2(제곱-제곱근)
# 입력:실수 a
# 출력:a의 절댓값

def abs_square(a):
    b = a * a
    return math.sqrt(b) # 수학 모듈의 제곱근 함수
	
print(abs_sign(5))
print(abs_sign(-3))
print(abs_square(5))
print(abs_square(-3))

=====
5
3
5.0
3.0
```
* 아래 제곱근 함수를 사용한 결과값에 소수점이 붙어 출력된 이유는 제곱근 함수가 소수점이 붙은 값을 돌려주기 때문
